#  脚本说明

 
### 配置文件
- `.env`: 环境变量配置文件
  ```
  OPENROUTER_API_KEY=""
  OPENROUTER_BASE_URL=""
  ```


## 环境设置

1. 创建并激活虚拟环境：
   ```
   python3 -m venv myenv
   source myenv/bin/activate
   ```
2. 安装依赖：
   ```
   pip install -r requirements.txt
   ```

## 运行

在 myenv 环境下 执行 

```
python publish_article.py
```
